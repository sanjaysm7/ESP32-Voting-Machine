# 🗳️ ESP32 Electronic Voting Machine

## 📌 Overview

This project implements an Electronic Voting Machine using ESP32 microcontroller with LCD display and push buttons.

## 👨‍💻 Team Members

* Sanjay SM
* Vignesh SB
* Adit Ranganathan Suresh

## ⚙️ Features

* Accurate vote counting
* LCD display interface
* Reset functionality
* Result display system
* Button debouncing

## 🧠 Technologies Used

* ESP32
* Arduino IDE
* Embedded C
